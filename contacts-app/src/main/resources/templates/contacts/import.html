Import contacts

Upload a CSV file. Import is based on the header row (first row).
← Back to contacts
Important: the first row must contain column names (headers).
Column order does not matter.

Required: email (used to match and update existing contacts).
Supported columns:
title, firstName, lastName, gender, email, phone1, phone2, institution, faculty, studyDomain, persGroup, function, country, coilExp, mobilityFin

Notes:
  • createdAt / updatedAt are ignored (auditing will set them).
  • coilExp and mobilityFin accept true/false, 1/0, yes/no.
  • Unknown extra columns are ignored.
